## 9. Permissions and Their Use

The App requests the following Android permissions. Here's exactly how each is used:

### 9.1 Required Permissions

**INTERNET** (`android.permission.INTERNET`)
- **Purpose**: Browse websites (core browser functionality)
- **When used**: Always (essential for a web browser)
- **Privacy**: Cannot be revoked for a browser to function

**NETWORK_STATE** (`android.permission.ACCESS_NETWORK_STATE`)
- **Purpose**: Detect internet connectivity
- **When used**: To show "No internet" message
- **Privacy**: Only checks if you're online, no data collection

**STORAGE** (`android.permission.READ/WRITE_EXTERNAL_STORAGE`)
- **Purpose**: Download and save files from websites
- **When used**: Only when you download or upload files
- **Privacy**: We only access files you explicitly choose

### 9.2 Optional Permissions (Requested Only When Needed)

**CAMERA** (`android.permission.CAMERA`)
- **Purpose**: Allow websites to access camera (e.g., QR code scanning, video calls)
- **When requested**: Only when a website specifically needs camera access
- **Privacy**: You must grant permission twice (Android system + website)
- **Control**: Revoke anytime in Android Settings
- **Our access**: We never access camera ourselves, only pass through to websites

**MICROPHONE** (`android.permission.RECORD_AUDIO`)
- **Purpose**: Allow websites to use microphone (e.g., voice calls, voice commands)
- **When requested**: Only when a website needs audio input
- **Privacy**: Explicit permission required each time
- **Our access**: We never record audio ourselves

**LOCATION** (`android.permission.ACCESS_COARSE_LOCATION`)
- **Purpose**: Allow websites to request approximate location (e.g., maps, weather)
- **When requested**: Only when a website asks for location
- **Privacy**: You control through browser permission prompts
- **Data collected**: Only approximate region (Firebase Analytics uses country/region)
- **Precision**: We never request ACCESS_FINE_LOCATION (precise GPS)

**NOTIFICATIONS** (`android.permission.POST_NOTIFICATIONS`)
- **Purpose**: Show app update notifications
- **When used**: For important announcements only
- **Control**: Disable in Settings anytime

### Permission Principles:

‚úÖ **Minimal Permissions**: We request only what's necessary
‚úÖ **Clear Explanations**: You know exactly why each permission is needed
‚úÖ **User Control**: Most permissions are optional and can be denied
‚úÖ **No Secret Access**: Permissions are used only for stated purposes
‚úÖ **Website Passthrough**: Camera/mic/location permissions are for websites, not us
